# Strategy Game (Unity)

A simple strategy game built in Unity, focusing on object placement, interaction, and gameplay mechanics.

## 🎮 About the Project

This project is a prototype of a strategy-style game where the player can place and interact with objects in a 3D environment.
The main focus is on gameplay logic, object placement systems, and basic interaction mechanics.
The project is built using Unity and C#.

---

## 🧩 Features

- Object placement system on valid surfaces
- Mouse-based interaction
- Rotation control for placed objects
- Modular script structure for gameplay logic
- Extensible system for future mechanics (AI, resources, etc.)
